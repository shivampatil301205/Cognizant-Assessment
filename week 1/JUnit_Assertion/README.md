# JUnit Assertions Exercise

Author: Shivam Patil

## What this project does
Demonstrates common JUnit 4 assertions using a simple test class.

## Project Structure
- `pom.xml` declares JUnit 4.13.2 dependency
- `src/test/java/AssertionsTest.java` contains the assertion examples
- `run.py` runs tests

## How to Run
From this folder:

```bash
python run.py
```
