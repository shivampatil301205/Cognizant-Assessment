# JUnit Mocking and Stubbing Exercise

This project demonstrates Mocking and Stubbing with Mockito and JUnit 5 (JUnit Jupiter).

## Scenario
You need to test a service (`MyService`) that depends on an external API (`ExternalApi`). Mockito is used to mock the external API and stub its methods.

## Steps
1. Create a mock object for the external API.
2. Stub the methods to return predefined values.
3. Write a test case that uses the mock object.

## How to Run
Run tests using:
```bash
python run.py
```
or
```bash
mvn test
```
