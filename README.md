# Cognizant Deep Skilling Program - Hands-on Submissions

This repository hosts all the hands-on coding submissions completed during the Cognizant Deep Skilling Program. The exercises are organized week-by-week. Each exercise folder contains its source code and an individual `README.md` with detailed explanations, setup instructions, and execution outputs/screenshots.

---

## Table of Contents (Week-Wise)

### [Week 1](week%201/)

#### 1. Software Design Patterns
*   [Factory Method Pattern](./Week%201/FactoryMethod%20Implementation/) - Decoupling object creation logic from clients.
*   [Singleton Pattern](./Week%201/Singleton%20Pattern%20Implementation/) - Ensuring a single shared class instance.

#### 2. Java Collections & Algorithms
*   [E-Commerce Platform](./Week%201/EcommercePlatform/) - Products, cart, inventory, and order management using Java Collections.
*   [Financial Forecasting](./Week%201/FinancialForecasting%20Ex/) - Recursion, memoization, and iteration algorithms comparison.

#### 3. Database Management (MySQL)
*   [PL/SQL Control Structures](./Week%201/ControlStructures/) - Banking operations, VIP promotions, and loan reminders.
*   [Stored Procedures](./Week%201/StoredProcedure/) - Automated restocking, salary updates, and issuing systems in library management.

#### 4. Unit Testing (JUnit & Mockito)
*   [JUnit Basic Testing](./Week%201/JUnit_BasicTesting/) - Basic unit testing fixtures (`@Before`/`@After`) and assertion methods.
*   [JUnit Assertions Exercise](./Week%201/JUnit_Assertion/) - Exploring different assertion types.
*   [JUnit AAA Pattern](./Week%201/JUnit_AAA/) - Structuring clean tests using the Arrange-Act-Assert convention.
*   [JUnit Mocking & Stubbing](./Week%201/JUnit_Mockito/) - Mocking external dependencies and stubbing methods using Mockito.
*   [JUnit Verifing Interactions](./Week%201/Mockito_Verify_Interaction/) - Verifying that methods are called with the correct arguments.

#### 5. SL4J Logging
*   [Logging Mrror Messages and Warning Level](./Week%201/Logging_Error_Warning/) - Demonstrating logging error messages and warning levels using SLF4J.

### [Week 2](./Week%202/)

#### 1. Spring Core_Maven
*   [SCM_SpringApplication](./Week%202/SCM_SpringApplication) - Building and running a basic Spring Core application using Maven.
*   [SCM_DependencyInjection](./Week%202/SCM_DependencyInjection) - Demonstrating Dependency Injection (DI) in Spring Core using XML-based configuration.
*   [SCM_MavenProject](./Week%202/SCM_MavenProject) - Creating and managing a Maven project with dependencies and build configuration.
*   [SCM_Spring_IoC_Container](./Week%202/SCM_Spring_IoC_Container) - Exploring the Spring IoC Container.
*   [SCM_Constructor_Setter_Injections](./Week%202/SCM_Constructor_Setter_Injections) - Demonstrating constructor and setter injection in Spring Core.
*   [SCM_SpringBootApplication](./Week%202/SCM_SpringBootApplication) - Building and running a Spring Boot application.

#### 2. Spring Data JPA with Spring Boot, Hibernate.
*   [orm-learn](./Week%202/orm-learn) - Developed a Spring Boot application using Spring Data JPA and MySQL to perform database operations on a Country entity.
*   [Difference(JPA,Hibernate,SpringDataJPA)](./Week%202/DIfference(JPA,Hibernate,SpringDataJPA)) - A concise comparison of JPA, Hibernate, and Spring Data JPA, explaining their roles, features, and how they simplify Java database persistence.
